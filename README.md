# Gitlytics Buddy

Gitlytics Buddy is a small web application that provides insights about GitHub repositories.  
It fetches repository information from the GitHub API and displays useful statistics such as commits, stars, forks, and other activity data in a simple dashboard.

This project was built as a practice project to explore working with APIs and building modern frontend applications.

---

## Tech Stack

This project uses the following technologies:

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui components

---

## Features

- Search for a GitHub repository
- View repository statistics
- Display commit and activity information
- Clean and responsive UI

## Running the Project Locally

Make sure you have **Node.js and npm installed**.

Clone the repository:
git clone https://github.com/YOUR_USERNAME/gitlytics-buddy.git


Go to the project directory:


cd gitlytics-buddy


Install dependencies:


npm install


Start the development server:


npm run dev


The app will run at:


http://localhost:5173
